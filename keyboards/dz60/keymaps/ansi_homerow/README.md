Ansi Homerow Layout
===

The goal of this layout is to maximize efficiency by keeping as much of the keyboard functionality on the home row, over multiple easily accessible layers. This layout relies on the 3-way split spacebar (either [2.75]-[1.25]-[2.25] or [2.25]-[1.25]-[2.75]). The main Space-key can be set to right or left, depending on personal preference (usually the [2.75] key).

---

BASE (Layer 0):
---
Nothing too crazy here. This is a Standard ANSI layer, save for 2 exception: The spacebar is split 3-ways, and the AllCaps key is replaced by a 
```
 /* BASE LAYER
 *
 * ,-----------------------------------------------------------------------------------------.
 * | Esc |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |  9  |  0  |  -  |  =  |  BACKSP   |
 * |-----------------------------------------------------------------------------------------+
 * | Tab    |  Q  |  W  |  E  |  R  |  T  |  Y  |  U  |  I  |  O  |  P  |  [  |  ]  |    \   |
 * |-----------------------------------------------------------------------------------------+
 * | MOD(2)  |  A  |  S  |  D  |  F  |  G  |  H  |  J  |  K  |  L  |  ;  |  '  |    Enter    |
 * |-----------------------------------------------------------------------------------------+
 * | Shift     |  Z  |  X  |  C  |  V  |  B  |  N  |  M  |  ,  |  .  |  /  |     RShift      |
 * |-----------------------------------------------------------------------------------------+
 * | Ctrl | Win | Alt |      Space       | MOD(3) | MOD(1) / Sp  | Alt | Menu | Func | Ctrl  |
 * `-----------------------------------------------------------------------------------------'
 */
 ```
 
LAYER 1:
 ---
 
This layer is meant to access the main function layer (F1-F12), as well as a lowered numrow, and other functionality. Note the Reset (used to flash the keyboard) on the right-Control key.
 
 ```
 /* LAYER 1
 *
 * ,-----------------------------------------------------------------------------------------.
 * |     |  F1 |  F2 |  F3 |  F4 |  F5 |  F6 |  F7 |  F8 |  F9 | F10 | F11 | F12 |           |
 * |-----------------------------------------------------------------------------------------+
 * |        |  1  |  2  |  3  |     |     |     |     |     |     |     |     |     |        |
 * |-----------------------------------------------------------------------------------------+
 * |         |     |     |     |     |     |     |     |     |     |     |     |             |
 * |-----------------------------------------------------------------------------------------+
 * |           |     |     |     |     |     |     |     |     |     |     |                 |
 * |-----------------------------------------------------------------------------------------+
 * |      |     |     |                  |        | oooooooooooo |     |      |      | RESET |
 * `-----------------------------------------------------------------------------------------'
 */
 ```

LAYER 2:
---
This row adds 2 clusters on the home-row, the arrow cluster under F, and the Delete-cluster under J. This layer is accessible from the standard CAPS key, making the arrow cluster accessible using only the left hand.

```
 /* LAYER 2
 *
 * ,-----------------------------------------------------------------------------------------.
 * |     |     |     |     |     |     |     |     |     |     |     |     |     |           |
 * |-----------------------------------------------------------------------------------------+
 * |        |     |     |  UP |     |     |     | Ins | Home| PgUp|     |     |     |        |
 * |-----------------------------------------------------------------------------------------+
 * | ooooooo |     | LEFT| DOWN|RIGHT|     |     | Del | End | PgDn|     |     |             |
 * |-----------------------------------------------------------------------------------------+
 * |           |     |     |     |     |     |     |     |     |     |     |                 |
 * |-----------------------------------------------------------------------------------------+
 * |      |     |     |                  |        |              |     |      |      |       |
 * `-----------------------------------------------------------------------------------------'
 */
 ```
 
LAYER 3:
 ---
 The middle-thumb function key enables quick desktop switching. The left-hand cluster under F can also be used for alternative functions
 
 ```
 /* LAYER 3
 *
 * ,-----------------------------------------------------------------------------------------.
 * |     |     |     |     |     |     |     |     |     |     |     |     |     |           |
 * |-----------------------------------------------------------------------------------------+
 * |        |     |     |     |     |     |     |     |     |     |     |     |     |        |
 * |-----------------------------------------------------------------------------------------+
 * |         |     |     |     |     |     |     |Dk_l |     |Dk_r |     |     |             |
 * |-----------------------------------------------------------------------------------------+
 * |           |     |     |     |     |     |     |     |     |     |     |                 |
 * |-----------------------------------------------------------------------------------------+
 * |      |     |     |                  | oooooo |              |     |      |      |       |
 * `-----------------------------------------------------------------------------------------'
 */
 ```
  
LAYER 4:
 ---

